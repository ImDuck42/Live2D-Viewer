Live2D Viewer Version 4.0 Released!

May 17, 2025

We are thrilled to announce the launch of Live2D Viewer version 4.0! This major update introduces a new way to discover and load models, alongside internal improvements.

  • New Feature: GitHub Repository Explorer!
      You can now directly browse GitHub repositories for Live2D models using the new File Explorer.
      Access it via the folder icon located to the left of the model URL input field.
      Enter any GitHub repository owner and name to explore its contents. A default repository, rich with numerous Live2D models, is pre-filled for convenience.
      (Please note: While this repository offers a wide selection, compatibility with every model is not guaranteed.)
  • Core Enhancements:
      Introduced new core files: js/repoexplorer.js and css/repoexplorer.css to power the File Explorer functionality.
      Internal file structure has been reorganized for better project management (e.g., some assets moved to an aaMisc directory).

Big Plans for the Website

May 16, 2025

  • Exciting News: A New Way to Explore Live2D Models is Coming!

    We're working on a fantastic update for you: a built-in File Explorer for our website!
    This new feature will let you navigate through all the files and folders within Live2D model hosting repositories, just like you would on your own computer.
    We believe this will greatly improve how you find and select your favorite models.
    Here's the really cool part: while you're browsing, if you spot a file that looks like a model's main configuration (it will usually have "model" in its name and a ".json" extension),
    you can import it directly with a single click, right from the explorer!
